# Revdev

revdev is a ruby binding to handling event devices.

## Installation

Add this line to your application's Gemfile:

```rb
gem 'revdev'
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install revdev
```
